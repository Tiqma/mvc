Prerequisites
----------------------------

Install php in terminal and Composer.

```
php install
composer install
```


Clone git
----------------------------

git clone https://github.com/Tiqma/Mvc


npm
----------------------------

```
npm install
```

```
npm run build
```

server
----------------------------

```
# You are in the app/ directory
php -S localhost:8888 -t public
```
